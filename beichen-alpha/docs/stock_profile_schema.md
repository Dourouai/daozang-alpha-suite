# 股票画像分类 Schema

股票画像不要只用一个 `themes` 字段。现在保留旧字段兼容，但推荐逐步拆成四类：

| 字段 | 作用 | 示例 |
|---|---|---|
| `primary_industry` | 主行业，只能有一个，用来做行业轮动和板块共振 | 工业金属、银行、半导体、公用事业 |
| `secondary_industries` | 辅行业/细分品种，可以多个 | 铜、铝、稀土、钼、光模块、新能源电池 |
| `style_tags` | 风格标签，用来做风格偏向，不参与行业轮动 | 高股息、防御、金融稳定、全球竞争、现金流 |
| `concept_tags` | 产业概念/主题，用来做产业链传导和观点匹配 | 新材料、国产替代、AI硬件、先进制造、机器人 |
| `themes` | 旧兼容字段，会自动拆分到上面三类 | 旧配置仍可继续使用 |

## 推荐 CSV

`config/profile_overrides.csv` 仍支持旧格式：

```csv
code,name,industry,themes,market_cap_billion
603993,洛阳钼业,工业金属,工业金属;资源;新材料;全球竞争,
```

也支持新格式：

```csv
code,name,industry,themes,market_cap_billion,primary_industry,secondary_industries,style_tags,concept_tags
603993,洛阳钼业,工业金属,工业金属;资源;新材料;全球竞争,3962,工业金属,钼;铜,全球竞争,新材料;全球竞争
600900,长江电力,公用事业,高股息;防御;能源安全;现金流,6619,公用事业,,高股息;防御;能源安全;现金流,
688981,中芯国际,半导体,半导体;国产替代;AI硬件,6500,半导体,,全球竞争,国产替代;AI硬件
```

## 使用规则

- 行业轮动只看 `primary_industry`、`secondary_industries`、部分 `concept_tags`。
- 风格偏向只看 `style_tags` 和少量产业概念，不再把“高股息/防御”混进行业共振。
- 宏观事件会映射到主/辅行业和产业概念，例如 `有色/铜/铝` 映射到 `工业金属`。
- 观点源会用全标签匹配，但消费/品牌消费等排除标签仍会一票过滤。

## 工业金属示例

| 股票 | 主行业 | 辅行业 | 风格 | 概念 |
|---|---|---|---|---|
| 洛阳钼业 | 工业金属 | 钼、铜 | 全球竞争 | 新材料 |
| 云南铜业 | 工业金属 | 铜 | 顺周期 | 资源 |
| 北方稀土 | 工业金属 | 稀土 | 资源周期 | 新材料 |
| 中国铝业 | 工业金属 | 铝 | 顺周期 | 资源 |

工业金属需要单独看，不要和黄金、煤炭、石油完全混成 `资源`。它更依赖美元、美债、人民币、铜铝价格、全球制造业、国内稳增长和新能源/AI材料需求。
